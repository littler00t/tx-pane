# `tx-pane` Compaction Samples

Auto-generated by `scripts/generate_compaction_samples.py`.
Before/after pairs for every builtin normalizer in `tx_compact/`.

**Generated inside the tx-tests Docker container** (Debian bookworm). Most samples are real `tx-pane run` output captured against the installed Linux binaries; the rest are captured fixtures from Linux servers where the container can't run the tool (no PID 1 systemd, no ZFS kernel module, no libvirtd, etc.). Source is labelled per section.

Each section shows:
- The shell command an agent would issue.
- **Raw** — what `tx-pane run --raw <pane> '<cmd>'` returns (cleaned bytes verbatim).
- **Compacted** — what `tx-pane run --terse <pane> '<cmd>'` returns.
- Byte-savings + which compaction layers fired.

**If a normalizer dropped information you need**, re-run the single command with `--no-normalize` (keep L1-L5) or `--raw` (escape hatch). See `docs/tx-doc-compaction.md` → "When compaction strips info you need".
